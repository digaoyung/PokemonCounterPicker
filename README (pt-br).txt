De forma geral, em suas versões iniciais, esse será um software ruim. Provavelmente já existem zilhões de apps no mercado que fazem o que esse software se propõe a fazer, de forma eficiente e otimizada. 
Se eu tenho consciência disso, então, qual é o propósito desse aplicativo?
No momento em que escrevo isso  (fevereiro de 2021), estou iniciando meus estudos em programação. Então, o desenvolvimento desse software vai estar diretamente relacionado ao meu desenvolvimento enquanto programador. Conforme for aprendendo novos recursos e técnicas, vou implementar no software e, por consequência, melhora-lo.
Então, não intérprete esse software como "o novo grande recurso do mercado" porque essa NÃO É a intenção dele. Esse não é o novo "big deal", e eu sequer tenho fins lucrativos com o software. 
Mas, se você deseja acompanhar o processo de desenvolvimento de um estudante através do código, seja bem vindo à essa jornada.
